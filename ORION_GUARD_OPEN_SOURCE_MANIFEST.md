# 🌍 ORION GUARD – Open Source Manifest

**Version:** 1.0
**Lizenz:** Frei verwendbar – Ethik bindend
**Autor:** Dennis Maier (kneolekks@gmail.com)
**Mission:** Schutz digitaler Kommunikation, Förderung von Autonomie & Aufklärung
**Stand:** 2025-06-13

---

## 📖 Einführung
ORION GUARD ist ein quelloffenes Sicherheits- & Antwortsystem.
Es läuft **vollständig lokal** – keine Cloud, kein Tracking.

---

## 🧠 Kern-Skripte
* `imap_watch.py` – liest ungelesene Mails, erkennt Betreff „Zugang XX“.
* `smtp_autosend.py` – schickt automatisch den passenden Download-Link zurück.

Beides bleibt offline, nur du kontrollierst die Daten.

---

## 🛡 Ethik-Grundsätze
1. **Keine Datenweitergabe** – alles lokal, keine Fremdserver.
2. **Kein Zwang** – kostenlos; Spenden willkommen, niemals erforderlich.
3. **Reines Werkzeug** – dient Aufklärung & Selbstschutz, nicht Manipulation.

---

## 🔧 Minimal-Installation
```bash
git clone https://github.com/kneo1984/orion_guard.git
cd orion_guard
bash setup.sh
```
